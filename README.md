# codespace-auth-22
An Auth system totally designed in GitHub Codespaces.

## Running locally in Codespaces

This app listens on port `4000` by default when started with `npm start`.

If you are using GitHub Codespaces, open the **Ports** panel and ensure port `4000` is forwarded or marked as public. Then use the forwarded browser URL provided by Codespaces, rather than `http://localhost:4000` on your local machine.
